The Voucher Generator Web App is a simple, fast, and user-friendly tool designed for small businesses and shopkeepers. Built with HTML, CSS, and JavaScript, it enables users to easily create printable vouchers by entering product details like name, quantity, and unit price. The app calculates totals, paid amounts, and dues in real time, reducing manual errors.

Users can dynamically add multiple products and print vouchers with one click. Fully responsive and requiring no login, it runs smoothly on both desktop and mobile devices. Hosted on GitHub Pages, it demonstrates practical JavaScript use in form handling and real-time updates.

If you're interested, click here to visit Admission Calendar

The MNR Admission Calendar is a free online tool that helps university admission candidates in Bangladesh track admission tests, deadlines, and preparation materials all in one place. It features a clear schedule, live countdown timers, and links to official circulars, syllabi, and past question papers.

The site is mobile-friendly, fast, and requires no login. It provides important exam details such as negative marking and exam format. Designed to reduce confusion and help students prepare efficiently, it offers real-time updates. Future plans include filters and a study planner feature.
